SmiFlavor Fields

The SmiFlavor type exposes the following members.

Fields
  NameDescription
Absolute
Output canonical SMILES with stereochemistry, atomic masses.
AtomAtomMap
Output atom-atom mapping for reactions and atom classes for molecules. The map index is set on an atom with property {@link org.openscience.cdk.CDKConstants#ATOM_ATOM_MAPPING} using {@link org.openscience.cdk.interfaces.IAtom#setProperty(Object, Object)}.
AtomicMass
Output atomic mass on atoms.
Canonical
Output SMILES in a canonical order. The order is not guaranteed to be equivalent between releases.
Cx2dCoordinates
Output 2D coordinates.
Cx3dCoordinates
Output 3D coordinates.
CxAtomLabel
Output atom labels, atom labels are specified by [!:IPseudoAtom.GetLabel()].
CxAtomValue
Output atom values, atom values are specified by [!:IPseudoAtom.GetLabel()].
CxCoordinates
Output either 2D/3D coordinates.
CxFragmentGroup
Output fragment grouping for reactions.
CxMulticenter
Output multicenter bonds, positional variation is specified with {@link org.openscience.cdk.sgroup.Sgroup}s of the type {@link org.openscience.cdk.sgroup.SgroupType#ExtMulticenter}.
CxPolymer
Output polymer repeat units is specified with {@link org.openscience.cdk.sgroup.Sgroup}s.
CxRadical
Output radicals, radicals are specified by [!:IAtomContainer.GetConnectedSingleElectronsCount(IAtom)]
CxSmiles
Output CXSMILES layers.
CxSmilesWithCoords
Output CXSMILES layers and coordinates.
Default
Default SMILES output write Stereochemistry, Atomic Mass, and CXSMILES layers. The ordering is not canonical.
Generic
Output non-canonical SMILES without stereochemistry, atomic masses.
InChILabelling
Output SMILES in a canonical order using the InChI labelling algorithm.
Isomeric
Output non-canonical SMILES with stereochemistry, atomic masses.

Output canonical SMILES with stereochemistry, atomic masses using the InChI labelling algorithm {@cite OBoyle12}. With delocalised charges the generated SMILES can be non-canonical.
