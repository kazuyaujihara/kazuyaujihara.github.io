Remarks
Any existing aromaticity flags are removed - even if no aromatic bonds were found. This follows the idea of applying an aromaticity model to a molecule such that the result is the same irrespective of existing aromatic flags. If you require aromatic flags to be preserved the FindBonds(IAtomContainer) can be used to find bonds without setting any flags.
