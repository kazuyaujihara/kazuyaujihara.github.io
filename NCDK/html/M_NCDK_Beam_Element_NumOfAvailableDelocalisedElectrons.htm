ElementNumOfAvailableDelocalisedElectrons Method

Determine the number of available electrons which could be bonding to implicit hydrogens for an aromatic atom with delocalized bonds. This include electrons donated from the hydrogen.

Remarks
Instead of checking higher valence states only the lowest is checked. For example nitrogen has valence 3 and 5 but in a delocalized system only the lowest (3) is used. The electrons which would allow bonding of implicit hydrogens in the higher valence states are donated to the aromatic system and thus cannot be reached. Using a generalisation that an aromatic bond as 3 electrons we reached the correct value for multi valence aromatic elements.

c1c[nH]cn1 the aromatic subset nitrogen is bonded to two aromatic nitrogen bond Order sum of 3 (6 electrons) there are no implicit hydrogens c1cc2ccccn2c1 the nitrogen has three aromatic bond 4.5 bond Order (9 electrons) - as we only check the lowest valence (3 - 4.5) < 0 so there are 0 implicit hydrogens c1ccpcc1 the phosphorus has 2 aromatic bond (bond Order sum 3) and the lowest valence is '3' - there are no implicit hydrogens oc1ccscc1 the sulphur has two aromatic bonds (bond Order sum 3) the lowest valence is '2' - 3 > 2 so there are no implicit hydrogens oc1ccscc1 the oxygen has a single aromatic bond, the default valence of oxygen in the specification is '2' there are no hydrogens (2 - 1.5 = 0.5).
