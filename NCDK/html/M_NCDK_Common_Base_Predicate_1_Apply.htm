Apply Method

PredicateTApply Method

Returns the result of applying this predicate to input. This method is generally expected, but not absolutely required, to have the following properties:
  • Its execution does not cause any observable side effects.
  • The computation is consistent with equals; that is, Equals(Object, Object) implies that predicate.Apply(a) == predicate.Apply(b).
