IBitFingerprintOr Method

Performs a logical OR of the bits in this target bit set with the bits in the argument fingerprint. This operation can also be seen as merging two fingerprints. This fingerprint is modified so that each bit in it has the value if and only if it either already had the value or the corresponding bit in the bit set argument has the value .
