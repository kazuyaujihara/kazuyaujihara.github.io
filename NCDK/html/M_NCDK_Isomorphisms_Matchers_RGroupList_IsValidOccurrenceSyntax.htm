RGroupListIsValidOccurrenceSyntax Method

Validates the occurrence value.
  • n : exactly n ;
  • n - m : n through m ;
  • > n : greater than n ;
  • < n : fewer than n ;
  • default (blank) is > 0 ;
Any non-contradictory combination of the preceding values is also allowed; for example "1, 3-7, 9, >11".
