SMARTSAtomChiralityMatches Method

Check if the atom-based chirality of the target matches. This check is done post-matching and should only be checked on atoms which are know to have already been matched (Matches(IAtom)). Currently the only atom-based chirality allowed is tetrahedral stereo-chemistry.
