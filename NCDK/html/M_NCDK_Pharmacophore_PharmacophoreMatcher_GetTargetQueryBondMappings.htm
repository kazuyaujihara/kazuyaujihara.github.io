PharmacophoreMatcherGetTargetQueryBondMappings Method

Return a list of HashMap's that allows one to get the query constraint for a given pharmacophore bond.

If the matching is successful, the return value is a List of HashMaps, each HashMap corresponding to a separate match. Each HashMap is keyed on the PharmacophoreBond in the target molecule that matched a constraint (PharmacophoreQueryBond or PharmacophoreQueryAngleBond. The value is the corresponding query bond.
