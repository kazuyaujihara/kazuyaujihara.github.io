DescriptorEngineCreateT Method

Create a descriptor engine for all descriptor types. Descriptors are loaded using the service provider mechanism. To include custom descriptors one should declare as an embedded resource named as the interface you are providing (e.g. IMolecularDescriptor). This file declares the implementations provided by the jar as class names.
