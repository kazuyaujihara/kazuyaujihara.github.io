DescriptorEngineGetDescriptorClassNameByPackage Method

Returns a list containing the classes found in the specified descriptor package.

The package name specified can be null or an empty string. In this case the package name is automatically set to "NCDK.QSAR.Descriptors" and as a result will return classes corresponding to both atomic and molecular descriptors.
