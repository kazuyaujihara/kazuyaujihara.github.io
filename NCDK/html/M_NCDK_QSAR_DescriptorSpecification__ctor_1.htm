Parameters

specificationReference
Type: SystemString
Reference to a formal definition in a dictionary (e.g. in STMML format) of the descriptor, preferably referring to the original article. The format of the content is expected to be <dictionaryNameSpace>:<entryID>.
implementationTitle
Type: SystemString
Title for the implementation of the descriptor for which the algorithm is given by the specification.
implementationIdentifier
Type: SystemString
Unique identifier for the actual implementation, preferably including the exact version number of the source code. E.g. $Id$ can be used when the source code is in a CVS repository.
implementationVendor
Type: SystemString
Name of the organisation/person/program/whatever who wrote/packaged the implementation.
