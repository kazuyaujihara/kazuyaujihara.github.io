Remarks
The symbol grammar allows easy construction from common symbols, for example:
Copy
new Atom("NH+");   // nitrogen cation with one hydrogen
new Atom("OH");    // hydroxy
new Atom("O-");    // oxygen anion
new Atom("13CH3"); // methyl isotope 13
                atom := {mass}? {symbol} {hcnt}? {fchg}?
                mass := \d+
                hcnt := 'H' \d+
                fchg := '+' \d+? | '-' \d+?
