StereoElementFactoryCreateExtendedTetrahedral Method

Create an extended tetrahedral element for the atom at index v. If an extended tetrahedral element could not be created then null is returned. An element can not be created if, one or more atoms was missing coordinates, the atom has an unspecified (wavy) bond, the atom is no non-planar bonds (i.e. up/down, wedge/hatch). The method does not check if tetrahedral chirality is supported - for this functionality use Stereocenters.
