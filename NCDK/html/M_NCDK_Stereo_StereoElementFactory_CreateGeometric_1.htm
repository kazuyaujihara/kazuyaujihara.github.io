StereoElementFactoryCreateGeometric Method (Int32, Int32, Stereocenters)

Create a geometric element (double-bond stereochemistry) for the provided atom indices. If the configuration could not be created a null element is returned. There is no configuration is the coordinates do not indicate a configuration, there were undefined coordinates or an unspecified bond label. The method does not check if double bond stereo is supported - for this functionality use Stereocenters.
