HOSECodeGeneratorGetHOSECode Method (IAtomContainer, IAtom, Int32)

Produces a HOSE code for Atom root in the IAtomContainerac. The HOSE code is produced for the number of spheres given by noOfSpheres. IMPORTANT: if you want aromaticity to be included in the code, you need to apply Apply(IAtomContainer)ac prior to using GetHOSECode(IAtomContainer, IAtom, Int32). This method only gives proper results if the molecule is fully saturated (if not, the order of the HOSE code might depend on atoms in higher spheres). This method is known to fail for protons sometimes.
