Methods
  NameDescription
AsAliphaticForm
Access an aliphatic form of this atom.
AsAromaticForm
Access an aromatic form of this atom. If the element can not be aromatic then the same atom is returned.
GetNumberOfHydrogens
(internal) The number of hydrogens this atom would have if it were vertex 'u' in the graph 'g'. If the atom is in the organic subset the value is computed - otherwise the labelled hydrogen count is returned.
IsAromatic
Whether this atom is aromatic.
