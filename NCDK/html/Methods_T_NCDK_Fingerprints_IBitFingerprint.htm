IBitFingerprint Methods

The IBitFingerprint type exposes the following members.

Methods
  NameDescription
And
Performs a logical AND of the bits in this target bit set with the bits in the argument fingerprint. This fingerprint is modified so that each bit in it has the value if and only if it both initially had the value and the corresponding bit in the fingerprint argument also had the value .
AsBitSet
Returns a BitArray representation of the fingerprint. This might take significantly more memory!
GetSetBits
Returns a listing of the bits in the fingerprint that are set to true.
Or
Performs a logical OR of the bits in this target bit set with the bits in the argument fingerprint. This operation can also be seen as merging two fingerprints. This fingerprint is modified so that each bit in it has the value if and only if it either already had the value or the corresponding bit in the bit set argument has the value .
Set
Sets the bit at the specified index to true. index
