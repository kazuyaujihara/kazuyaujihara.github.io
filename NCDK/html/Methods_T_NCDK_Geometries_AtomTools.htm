Methods
  NameDescription
Add3DCoordinates1
Generate coordinates for all atoms which are singly bonded and have no coordinates. This is useful when hydrogens are present but have no coordinates. It knows about C, O, N, S only and will give tetrahedral or trigonal geometry elsewhere. Bond lengths are computed from covalent radii if available. Angles are tetrahedral or trigonal

Calculate3DCoordinates1
Calculate new Point(s) X in a B-A system to form B-A-X. Use C as reference for * staggering about the B-A bond
Calculate3DCoordinates2
Calculate new Point(s) X in a B-A-C system. It forms form a B-A(-C)-X system.
Calculate3DCoordinates3
Calculate new point X in a B-A(-D)-C system. It forms a B-A(-D)(-C)-X system.
Calculate3DCoordinatesForLigands
Adds 3D coordinates for singly-bonded ligands of a reference atom (A). Initially designed for hydrogens. The ligands of refAtom are identified and those with 3D coordinates used to generate the new points. (This allows structures with partially known 3D coordinates to be used, as when groups are added.)
RescaleBondLength
Rescales Point2 so that length 1-2 is sum of covalent radii. if covalent radii cannot be found, use bond length of 1.0
