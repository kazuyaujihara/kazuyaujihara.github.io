GetAtomsTo(Int32)
Reconstruct a shortest path to the provided end vertex. The path is an inclusive fixed size array IAtoms. If there are multiple shortest paths the first shortest path is determined by vertex storage order. When there is no path an empty array is returned. It is considered there to be no path if the end vertex belongs to the same container but is a member of a different fragment, or the vertex is not present in the container at all.
GetAtomsTo(IAtom)
Reconstruct a shortest path to the provided end atom. The path is an inclusive fixed size array IAtoms. It is considered there to be no path if the end atom belongs to the same container but is a member of a different fragment, or the atom is not present in the container at all.
GetDistanceTo(Int32)
Access the distance to the provided end vertex. If the two are not connected the distance is returned as MaxValue. Formally, there is a path if the distance is less then the number of vertices.
GetDistanceTo(IAtom)
Access the distance to the provided end atom. If the two are not connected the distance is returned as MaxValue. Formally, there is a path if the distance is less then the number of atoms.

GetPathsTo(Int32)
Reconstruct all shortest paths to the provided end vertex. The paths are n (where n is GetNPathsTo(Int32)) inclusive fixed size arrays of vertex indices. When there is no path an empty array is returned. It is considered there to be no path if the end vertex belongs to the same container but is a member of a different fragment, or the vertex is not present in the container at all.

IsPrecedingPathTo
Returns whether the first shortest path from the start to a given end vertex which only passed through vertices smaller then start. This is useful for reducing the search space, the idea is used by Vismara, Philippe, Union of all the minimum cycle bases of a graph, Combinatorics, 1997, 4:?-? in the computation of cycle prototypes.
