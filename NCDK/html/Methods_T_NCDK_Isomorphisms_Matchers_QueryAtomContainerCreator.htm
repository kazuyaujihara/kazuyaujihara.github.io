The QueryAtomContainerCreator type exposes the following members.

Methods
  NameDescription
CreateAnyAtomAnyBondContainer
Creates a QueryAtomContainer with wildcard atoms and wildcard bonds.

This method thus allows the user to search based only on connectivity.

CreateAnyAtomContainer
Creates a QueryAtomContainer with AnyAtoms / Aromatic Atoms and OrderQueryBonds / AromaticQueryBonds. It uses the CDKConstants.ISAROMATIC flag to determine the aromaticity of container.
CreateAnyAtomForPseudoAtomQueryContainer
Creates a QueryAtomContainer with SymbolQueryAtom's and OrderQueryBond's. Each IPseudoAtom will be replaced by a AnyAtom
CreateBasicQueryContainer
Creates a QueryAtomContainer with SymbolQueryAtom's, AromaticQueryBond's and OrderQueryBond's. If a IBond of the input container is flagged aromatic, then it disregards bond order information and only match against an aromatic target atom instead.
CreateSymbolAndBondOrderQueryContainer
Creates a QueryAtomContainer with SymbolQueryAtom's and OrderQueryBond's. Unlike CreateBasicQueryContainer(IAtomContainer), it disregards aromaticity flags.
CreateSymbolAndChargeQueryContainer
Creates a QueryAtomContainer with SymbolAndChargeQueryAtom's and OrderQueryBond's.
