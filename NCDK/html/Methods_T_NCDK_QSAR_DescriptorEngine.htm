CreateT
Create a descriptor engine for all descriptor types. Descriptors are loaded using the service provider mechanism. To include custom descriptors one should declare as an embedded resource named as the interface you are providing (e.g. IMolecularDescriptor). This file declares the implementations provided by the jar as class names.

GetDescriptorClassNameByPackage
Returns a list containing the classes found in the specified descriptor package.

The package name specified can be null or an empty string. In this case the package name is automatically set to "NCDK.QSAR.Descriptors" and as a result will return classes corresponding to both atomic and molecular descriptors.
