IDescriptor Methods

The IDescriptor type exposes the following members.

Methods
  NameDescription
GetParameterType
Returns a class matching that of the parameter with the given name. May only return for when name does not match any parameters returned by the Parameters method.
Initialise
Initialise the descriptor with the specified chem object builder. This allows descriptors that required domain objects, such as for SMARTS queries to initialise correctly. If you do not need domain objects then this method does not need to be implemented.
