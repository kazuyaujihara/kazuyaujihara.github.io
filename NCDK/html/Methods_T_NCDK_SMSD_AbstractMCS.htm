GetAllAtomMapping
Returns all plausible mappings between query and target molecules Each map in the list has atom-atom equivalence of the mappings between query and target molecule i.e. map.Key for the query and map.Value for the target molecule.
GetAllMapping
Returns all plausible mappings between query and target molecules Each map in the list has atom-atom equivalence index of the mappings between query and target molecule i.e. map.Key for the query and map.Value for the target molecule.
GetEnergyScore
Returns summation energy score of the disorder if the MCS is removed from the target and query graph. Amongst the solutions, a solution with lowest energy score is preferred.
GetEuclideanDistance
Returns Euclidean Distance between query and target molecule.

Return Value
