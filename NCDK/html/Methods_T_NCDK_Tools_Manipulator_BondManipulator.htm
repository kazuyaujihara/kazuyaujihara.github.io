IsHigherOrder
Returns true if the first bond has a higher bond order than the second bond. It returns false if the bond order is equal, and if the order of the first bond is lower than that of the second. Also returns false if either bond order is unset.
IsLowerOrder
Returns true if the first bond has a lower bond order than the second bond. It returns false if the bond order is equal, and if the order of the first bond is larger than that of the second. Also returns false if either bond order is unset.
