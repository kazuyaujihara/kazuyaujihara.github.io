XMLIsotopeFactory
Used to store and return data of a particular isotope. As this class is a singleton class, one gets an instance with:
Copy
IsotopeFactory ifac = IsotopFactory.GetInstance(new ChemObject().Builder);

Data about the isotopes are read from the file NCDK.Config.Data.isotopes.xml in the cdk-standard module. Part of the data in this file was collected from the website webelements.org.

The use of this class is exemplified as follows. To get information about the major isotope of hydrogen, one can use this code:

Copy
IsotopeFactory factory = XMLIsotopeFactory.GetInstance(DefaultChemObjectBuilder.Instance);
Isotope major = factory.GetMajorIsotope("H");
