Classes
  ClassDescription
AtomTools
A set of static utility classes for geometric calculations on Atoms.
BondTools
A set of static utility classes for geometric calculations on IBonds. The methods for detecting stereo configurations are described in CDK news, vol 2, p. 64 - 66.
CrystalGeometryTools
A set of static methods for working with crystal coordinates.
GeometryUtil
A set of static utility classes for geometric calculations and operations. This class is extensively used, for example, by JChemPaint to edit molecule. All methods in this class change the coordinates of the atoms. Use GeometryTools if you use an external set of coordinates (e. g. renderingCoordinates from RendererModel)
Projector
Tool to make projections from 3D to 2D.
RDFCalculator
Calculator of radial distribution functions. The RDF has bins defined around a point, i.e. the first bin starts at 0 Å and ends at 0.5*resolution Å, and the second bins ends at 1.5*resolution Å.
ZMatrixTools
A set of static utility classes for dealing with Z matrices.
