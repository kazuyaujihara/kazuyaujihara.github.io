SettingManagerT
Provides dynamic management of settings. This was created with the intention of managing IOSetting's for IChemObjectIO however it could be recycled for other purposes where dynamic settings are required. Settings are stored in a IDictionaryTKey, TValue using the name of the setting as the key. The name is normalised (lowercase and whitespace removal) to allow 'fuzzy' setting access. This means that character case differences do not affect the retrieval of objects.
