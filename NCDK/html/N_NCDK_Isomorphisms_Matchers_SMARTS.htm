Classes
  ClassDescription
AliphaticAtom
This matcher any non-aromatic atom. This assumes that aromaticity in the molecule has been perceived.
AliphaticSymbolAtom
This smarts atom matches aliphatic atom with element symbol specified.
AnyAtom
This matcher any Atom including explicit hydrogens.
AnyOrderQueryBond
This matches a bond of any order.
AromaticAtom
This matcher any aromatic atom. This assumes that aromaticity in the molecule has been perceived.
AromaticOrSingleQueryBond
This matches an aromatic or a single bond, used when no bond is specified between an atom.
AromaticQueryBond
This matches an aromatic bond.
AromaticSymbolAtom
This class matches aromatic atom with element symbol specified.
AtomicNumberAtom
This matches an atom using the atomic number.
ChiralityAtom
This matches an atom with chirality property. It is not implemented yet. It'll match any atom right now.
ExplicitConnectionAtom
Match an atom with the defined degree. The degree is also referred to as the explicit connectivity and is encoded in smarts using D<NUMBER>.
FormalChargeAtom
This matcher checks the formal charge of the Atom.
HybridizationNumberAtom
This matcher checks the hybridization state of an atom.
HydrogenAtom
This matches Hydrogen atoms.
ImplicitHCountAtom
This matcher checks the number of implicit hydrogens of the Atom.
LogicalOperatorAtom
This class matches a logical operator that connects two query atoms. Logical matchers are created with, And(IQueryAtom, IQueryAtom), Not(IQueryAtom) and Or(IQueryAtom, IQueryAtom).
LogicalOperatorBond
This class matches a logical operator that connects two query bonds.
MassAtom
This class matches an atom based on the atomic mass.
NonCHHeavyAtom
This matcher any heavy atom that is not C or H.
OrderQueryBond
This matches a bond with a certain bond order. Daylight spec indicates that if match a single bond using '-', it should be an aliphatic single bond
PeriodicGroupNumberAtom
This matcher checks the periodic group number of an atom.
RecursiveSmartsAtom
This matches recursive smarts atoms.
RingBond
This smarts bond matches any bond that is in a ring.
RingIdentifierAtom
This encapsulates an atom with a ring identifier, with an optional ring bond specified. For example, C=1CCCCC1.
RingMembershipAtom
This query is found in a specified number of ring. The ring membership is specified with the SMARTS R<NUMBER>. The membership depends on the ring set used and as such is not a portable term. If the Smallest Set of Smallest Rings (SSSR) is used then changing the order of atoms may change which atoms match in a pattern.
SmallestRingAtom
Match an atom in a specific size ring. The ring size is specified by r<NUMBER> in a SMARTS pattern. This term is non-portable, depending on the set of rings chosen and which ring sizes are used. The default implementation (Daylight) only stores the smallest ring each atom belongs to whilst other implementations may store multiple values. A more portable term is the ring connectivity which is specified as x<NUMBER>.
SMARTSAtom
Abstract smarts atom.
SMARTSBond
Abstract smarts bond.
SmartsMatchers
Bridging class between the SMARTS matcher and the parser/query tool. The class are currently split across different packages. This classes temporary functionality is to expose package private functionality through a single location.
StereoBond
This query bond indicates a particular geometric stereo configuration.
TotalConnectionAtom
This matches an atom using total number of connections - referred to in SMARTS as the connectivity. The connectivity is specified using the X<NUMBER> pattern.
TotalHCountAtom
SMARTS query atom for matching the total hydrogen count. This count is specified in SMARTS using H<NUMBER>.
TotalRingConnectionAtom
This matcher checks the number of ring connections of the checked Atom with other Atom's. This cannot be matched without prepossessing Atom - SMARTSAtomInvariants. The ring connectivity is encoded in smarts using x<NUMBER>.
TotalValencyAtom
This matcher checks the valence of the Atom. The valence is the number of bonds formed by an atom (including bonds to implicit hydrogens).
