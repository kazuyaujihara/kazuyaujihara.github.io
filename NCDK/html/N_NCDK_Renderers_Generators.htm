Classes

AtomContainerBoundsGenerator
IGeneratorT that draws a rectangular around the IAtomContainer.
AtomMassGenerator
IGeneratorT that can render mass number information of atoms.
AtomNumberGenerator
IGeneratorT for IAtomContainers that will draw atom numbers for the atoms.
BasicAtomGenerator
Generates basic IRenderingElements for atoms in an atom container.
BasicBondGenerator
Generator for elements from bonds. Only two-atom bonds are supported by this generator.
BasicGenerator
Combination generator for basic drawing of molecules. It only creates drawing elements for atoms and bonds, using the BasicAtomGenerator and BasicBondGenerator.
BasicSceneGenerator
This generator does not create any elements, but acts as a holding place for various generator parameters used by most drawings, such as the zoom, background color, margin, etc.
BoundsGenerator
Produce a bounding rectangle for various chem objects.
ExtendedAtomGenerator
A generator for atoms with mass, charge, etc.
HighlightGenerator
Generate an under/overlaid highlight in structure depictions. The highlight emphasises atoms and bonds. Each atom and bond is optionally assigned an integer identifier. Entities with identifiers are then highlighted using the IPalette to determine the color. The size of the highlight is specified with the GetHighlightRadius(RendererModel) parameter.
LonePairGenerator
Generate the symbols for lone pairs.
MappingGenerator
IGeneratorT that will show how atoms map between the reactant and product side.
ProductsBoxGenerator

RingGenerator
Generates just the aromatic indicators for rings : circles, or light-gray inner bonds, depending on the value of CDKStyleAromaticity.
