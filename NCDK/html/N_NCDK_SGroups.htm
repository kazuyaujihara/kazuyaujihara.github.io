DeduceBondSystemTool Obsolete.
Tool that tries to deduce bond orders based on connectivity and hybridization for a number of common ring systems of up to seven-membered rings. It assumes that atom types have been perceived before that class is used.

The calculation can be interrupted with Interrupted, but assumes that this class is not used in a threaded fashion. When a calculation is interrupted, the bool is reset to false.

FixBondOrdersTool
Class to Fix bond orders at present for Aromatic Rings only.
Sgroup
Generic CTab Sgroup (substructure group) that stores all other types of group. This representation is allows reading from CTfiles (e.g. Molfile, SDfile).
