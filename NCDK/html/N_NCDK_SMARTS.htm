SmartsFragmentExtractor
Utility class to create SMARTS that match part (substructure) of a molecule. SMARTS are generated by providing the atom indexes. An example use cases is encoding features from a fingerprint.
SmartsPattern
A Pattern for matching a single SMARTS query against multiple target compounds. The class should not be used for matching many queries against a single target as in substructure keyed fingerprints. The SMARTSQueryTool is currently a better option as less target initialistion is performed.
