BKKCKCF
This class implements Bron-Kerbosch clique detection algorithm as it is described in [F. Cazals, vertexOfCurrentClique. Karande: An Algorithm for reporting maximal c-cliques; processedVertex.Comp. Sc. (2005); vol 349; pp. 484-490] BronKerboschCazalsKarandeKochCliqueFinder.java
ExactMapping
This class handles MCS between two identical molecules. Hence they generate am MCS where all atoms are mapped.
GenerateCompatibilityGraph
This class generates compatibility graph between query and target molecule. It also markes edges in the compatibility graph as c-edges or d-edges.
MCSPlus Obsolete.
This class handles MCS plus algorithm which is a combination of c-clique algorithm and McGregor algorithm.
MCSPlusHandler Obsolete.
This class acts as a handler class for MCSPlus algorithm.
