GetBondLengthAverage(IAtomContainer)
An average of all 2D bond length values is produced. Bonds which have Atom's with no coordinates are disregarded. See comment for Center(IAtomContainer atomCon, Dimension areaDim, Dictionary renderingCoordinates) for details on coordinate sets

Returns the average 2D bond length values of all products and reactants of the given reaction. The method uses GetBondLengthAverage(IAtomContainer) internally.
