GetAutomorphismGroup(IAtomContainer)
Gets the automorphism group of the atom container. By default it uses an initial partition based on the bond 'types' (so all the C-C bonds are in one cell, all the C=N in another, etc). If this behaviour is not desired, then use the ignoreBondOrders flag in the constructor.
