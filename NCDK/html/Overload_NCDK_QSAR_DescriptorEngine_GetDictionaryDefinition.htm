GetDictionaryDefinition(String)
Gets the definition of the descriptor.

All descriptors in the descriptor dictioanry will have a definition element. This function returns the value of that element. Many descriptors also have a description element which is more detailed. However the value of these elements can contain arbitrary mark up (such as MathML) and I'm not sure what I should return it as A string containing either the descriptors fully qualified class name or else the descriptors specification reference
