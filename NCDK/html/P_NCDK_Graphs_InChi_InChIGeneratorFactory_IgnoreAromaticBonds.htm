Remarks
Sets whether aromatic bonds should be treated as single and double bonds for the InChI generation.The bond type INCHI_BOND_TYPE.Altern is considered special in contrast to single, double, and triple bonds, and is not bulletproof. If the molecule has clearly defined single and double bonds, the option can be used to force the class not to use the alternating bond type. http://www.inchi-trust.org/fileadmin/user_upload/html/inchifaq/inchi-faq.html#16.3
