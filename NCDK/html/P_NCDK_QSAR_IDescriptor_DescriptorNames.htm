IDescriptorDescriptorNames Property

Returns an array of names for each descriptor value calculated.

Namespace:  NCDK.QSAR
Assembly:  NCDK (in NCDK.dll) Version: 1.0.0.0 (1.0.0.0)
Syntax
C#
Copy
IReadOnlyList<string> DescriptorNames { get; }

Property Value

Type: IReadOnlyListString
An array of descriptor names, equal in length to the number of descriptor calculated.
Remarks

Many descriptors return multiple values. In general it is useful for the descriptor to indicate the names for each value.

In many cases, these names can be as simple as X1, X2, ..., XN where X is a prefix and 1, 2, ..., N are the indices. On the other hand it is also possible to return other arbitrary names, which should be documented in the Javadocs for the descriptor (e.g., the CPSA descriptor).

Note that by default if a descriptor returns a single value (such as ALOGPDescriptor the return array will have a single element
