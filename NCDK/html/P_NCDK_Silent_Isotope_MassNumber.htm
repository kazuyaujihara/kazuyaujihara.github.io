Examples
Once instantiated all field not filled by passing parameters to the constructor are null. Isotopes can be configured by using the Configure(IAtom, IIsotope) method:
Copy
IAtom atom = new Atom();
Isotope isotope = new Isotope("C", 13);
XMLIsotopeFactory f = XMLIsotopeFactory.GetInstance(isotope.Builder);
f.Configure(atom, isotope);
