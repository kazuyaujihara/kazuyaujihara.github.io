Atom Properties

The Atom type exposes the following members.

Properties
  NameDescription
AtomClass
The class of the atom is defined as an integer value. The atom class is specified for bracketed atoms and is prefixed by a colon.

[CH:1](C)([C])[H:2]

Charge
Formal charge of the atom.
Element
The element of the atom.
Isotope
The isotope number of the atom. If the isotope is undefined (default) a value -1 is returned.
Label
An label attached to an element (input only). Although invalid via the specification 'CCC[R]' etc can occur in the 'wild'. If found the parser provides an 'Unknown' element and a specified label. Not the labels are never written. By default the label is the element symbol.
NumOfHydrogens
Number of hydrogens this atom has. This value defines atoms with an explicit hydrogen count of bracket atoms (e.g. [CH4]).
Subset
(internal) Is the atom a member of the organic (aromatic/aliphatic) subset implementation?
Token
(internal) The token to write for the atom when generating a SMILES string.
