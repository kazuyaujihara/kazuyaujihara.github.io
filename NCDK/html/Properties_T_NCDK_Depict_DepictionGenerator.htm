DepictionGenerator Properties

The DepictionGenerator type exposes the following members.

Properties
  NameDescription
AlignMappedReaction
Specifies that reactions with atom-atom mappings should have their reactants/product coordinates aligned. Default: true.
AnnotateAtomMapNumbers
Display atom-atom mapping numbers on a reaction. Each atom map index is loaded from the property AtomAtomMapping.
AnnotateAtomNumbers
Display atom numbers on the molecule or reaction. The numbers are based on the ordering of atoms in the molecule data structure and not a systematic system such as IUPAC numbering.
AnnotateAtomValues
Display atom values on the molecule or reaction. The values need to be assigned by atom.SetProperty(CDKConstants.COMMENT, myValueToBeDisplayedNextToAtom);
AnnotationColor
The color annotations (e.g. atom-numbers) will appear in.
AnnotationFontScale
The size of annotations relative to atom symbols.
AtomColorer
Colorer for atom symbols.
AtomMapColors
Adds to the highlight the coloring of reaction atom-maps. The optional color array is used as the pallet with which to highlight. If none is provided a set of high-contrast colors will be used.
BackgroundColor
The background color.
FillToFit
Resize depictions to fill all available space (only if a size is specified). This generally isn't wanted as very small molecules (e.g. acetaldehyde) may become huge.
Highlighting
Margin
The desired size of margin. The units depend on the output format with raster images using pixels and vector graphics using millimeters.
OuterGlowWidth
Padding
The desired size of padding for molecule sets and reactions. The units depend on the output format with raster images using pixels and vector graphics using millimeters.
ShowMoleculeTitle
Display a molecule title with each depiction. The title is specified by setting the Title property. For reactions only the main components have their title displayed.
ShowReactionTitle
Display a reaction title with the depiction. The title is specified by setting the Title property on the IReaction instance.
Size
Specify a desired size of depiction. The units depend on the output format with raster images using pixels and vector graphics using millimeters. By default depictions are only ever made smaller if you would also like to make depictions fill all available space use the FillToFit option.
SymbolVisibility
Atom symbol visibility.
TitleColor
The color titles will appear in.
TitleFontScale
The size of titles compared to atom symbols.
Zoom
The desired zoom factor - this changes the base size of a depiction and is used for uniformly making depictions bigger. If you would like to simply fill all available space (not recommended) use FillToFit.
