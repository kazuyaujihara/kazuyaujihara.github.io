ElectronDonation Class

Defines an electron donation model for perceiving aromatic systems. The model defines which atoms are allowed and how many electron it contributes.

Remarks
There are currently several models available.
  • CDK/CDKAllowingExocyclicModel - uses the information form the preset CDK atom types to determine how many electrons each atom should contribute. The model can either allow or exclude contributions from exocyclic pi bonds. This model requires that atom types have be perceived.
  • PiBondsModel - a simple model only allowing cyclic pi bonds to contribute. This model only requires that bond orders are set.
  • DaylightModel - a model similar to that used by Daylight for SMILES. This model does not require atom types to be defined but every atom should have it's hydrogen count set.
Examples
To obtain an instance of the model simply invoke the named method.
Copy
ElectronDonation model = ElectronDonation.CDKModel;
