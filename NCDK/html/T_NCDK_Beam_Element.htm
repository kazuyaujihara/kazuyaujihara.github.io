Remarks

Organic subsets

Several of the elements belong to the organic subset. Atoms of an organic element type can be written just as their symbol (see. Organic Subset, OpenSMILES Specification).
  • Unknown (*)
  • Boron
  • Carbon
  • Nitrogen
  • Oxygen
  • Fluorine
  • Phosphorus
  • Sulfur
  • Chlorine
  • Bromine
  • Iodine

Examples

Usage

Elements can be created by either using the value directly or by looking up it's symbol. If the element may be aromatic the lower-case symbol can also be used. For example the variable 'e' in the three statements below all have the same value, Carbon.
Copy
Element e = Element.Carbon;
Element e = Element.OfSymbol("C");
Element e = Element.OfSymbol("c");
When the symbol is invalid the result wil be null.
Copy
Element e = Element.OfSymbol("R1"); // e = null
The Unknown element can be used to represent generic/alias atoms.
Copy
Element e = Element.Unknown;
Element e = Element.OfSymbol("*");
To access the symbol of an already created element. Use Symbol.
Copy
IAtom    a = ...;
Element e = a.Element;

string  symbol = e.Symbol;
