Differences
List all differences between the two bit vectors. Unlike ListDifferences(BitArray, BitArray) which only list those which are set in s but not in t.
IsSubset
Checks whether all the positive bits in BitArray bs2 occur in BitArray bs1. If so, the molecular structure from which bs2 was generated is a possible substructure of bs1.
ListDifferences
This lists all bits set in bs2 and not in bs2 (other way round not considered) in a list and to logger. See. Differences(BitArray, BitArray) for a method to list all differences, including those missing present in bs2 but not bs1.
