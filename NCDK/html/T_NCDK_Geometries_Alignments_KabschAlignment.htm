Remarks
This class is an implementation of the Kabsch algorithm (Kabsch, W., A Solution for the Best Rotation to Relate Two Sets of Vectors, Acta Cryst., 1976, A32:922-923, Kabsch, W., Discussion of Solution for Best Rotation to Relate 2 Sets of Vectors, Acta Cryst., 1978, A34:827-828) and evaluates the optimal rotation matrix (U) to minimize the RMSD between the two structures. Since the algorithm assumes that the number of points are the same in the two structures it is the job of the caller to pass the proper number of atoms from the two structures. Constructors which take whole IAtomContainer's are provided but they should have the same number of atoms. The algorithm allows for the use of atom weightings and by default all points are given a weight of 1.0,
Examples
Copy
var ac1 = new AtomContainer();    // molecule 1
var ac2 = new AtomContainer();    // molecule 2
try
{
    KabschAlignment sa = new KabschAlignment(ac1.Atoms, ac2.Atoms);

    sa.Align();
    Console.Out.WriteLine(sa.RMSD);
}
catch (CDKException) { }
In many cases, molecules will be aligned based on some common substructure. In this case the center of masses calculated during alignment refer to these substructures rather than the whole molecules. To superimpose the molecules for display, the second molecule must be rotated and translated by calling RotateAtomContainer(IAtomContainer). However, since this will also translate the second molecule, the first molecule should also be translated to the center of mass of the substructure specified for this molecule. This center of mass can be obtained by a call to CenterOfMass and then manually translating the coordinates. Thus an example would be
Copy
AtomContainer ac1 = new AtomContainer();    // whole molecules
AtomContainer ac2 = new AtomContainer();    // 
IAtom[] a1 = ac1.Atoms.ToArray();   // some subsets of atoms from the two molecules
IAtom[] a2 = ac2.Atoms.ToArray();   // 
try
{
    var sa = new KabschAlignment(a1, a2);
    sa.Align();

    var cm1 = sa.CenterOfMass;
    foreach (var a in ac1.Atoms)
        a.Point3D = a.Point3D.Value - cm1;
    sa.RotateAtomContainer(ac2);
    // display the two AtomContainer's
}
catch (CDKException) { }
