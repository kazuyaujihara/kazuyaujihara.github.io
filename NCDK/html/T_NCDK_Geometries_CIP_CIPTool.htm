CIPTool Class

Tool to help determine the R,S and stereochemistry definitions of a subset of the CIP rules Cahn, R.S. and Ingold, C. and Prelog, V., Specification of Molecular Chirality, Angew. Chem. Int. Ed., 1966, 5:385-415. The used set up sub rules are specified in the CIPLigandRule class.

Examples
Basic use starts from a ITetrahedralChirality and therefore assumes atoms with four neighbours:
Copy
IAtom[] ligandAtoms = mol.GetConnectedAtoms(centralAtom).ToArray();
ITetrahedralChirality tetraStereo = new TetrahedralChirality(centralAtom, ligandAtoms, TetrahedralStereo.AntiClockwise);
CIPChiralities cipChirality = CIPTool.GetCIPChirality(mol, tetraStereo);
The BondStereo value can be reconstructed from 3D coordinates with the StereoTool.
