RDFCalculator Class

Calculator of radial distribution functions. The RDF has bins defined around a point, i.e. the first bin starts at 0 Å and ends at 0.5*resolution Å, and the second bins ends at 1.5*resolution Å.

Examples
By default, the RDF is unweighted. By implementing and registering a RDFCalculatorWeightFunction, the RDF can become weighted. For example, to weight according to partial charge interaction, this code could be used:
Copy
RDFCalculator calculator = new RDFCalculator(0.0, 5.0, 0.1, 0.0,
    delegate(IAtom atom, IAtom atom2) { return atom.Charge.Value * atom2.Charge.Value; });
