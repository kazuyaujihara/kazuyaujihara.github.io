EnumerableSMILESReader Class

Iterating SMILES file reader. It allows to iterate over all molecules in the SMILES file, without being read into memory all. Suitable for very large SMILES files. These SMILES files are expected to have one molecule on each line. If a line could not be parsed and empty molecule is returned and the property BadSmilesInput is set to the attempted input. The error is also logged.

For parsing each SMILES it still uses the normal SMILESReader.
