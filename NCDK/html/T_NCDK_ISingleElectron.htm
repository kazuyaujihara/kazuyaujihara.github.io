Examples
A radical in CDK is represented by an IAtomContainer that contains an IAtom and a ISingleElectron type IElectronContainer:
Copy
AtomContainer radical = new AtomContainer();
Atom carbon = new Atom("C");
carbon.ImplicitHydrogenCount = 3;
radical.Add(new SingleElectron(carbon));
