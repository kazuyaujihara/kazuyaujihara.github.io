Find and count the number molecules which contain the query substructure.
Copy
IAtomContainer query = queryStructure;
Pattern pattern = Ullmann.FindSubstructure(query);

int hits = 0;
foreach (var m in ms)
    if (pattern.Matches(m))
        hits++;
Finding the matching to molecules which contain the query substructure. It is more efficient to obtain the Match(IAtomContainer) and check it's size rather than test if it MatchAll(IAtomContainer) first. These methods automatically verify stereochemistry.
Copy
IAtomContainer query = queryStructure;
Pattern pattern = Ullmann.FindSubstructure(query);

int hits = 0;
foreach (var m in ms)
{
    int[] match = pattern.Match(m);
    if (match.Length > 0)
        hits++;
}
