INCHI_KEY Enumeration

Type-safe enumeration of InChI return codes.

InChI library return values:

  • OK = 0
  • UNKNOWN_ERROR = 1
  • EMPTY_INPUT = 2
  • NOT_INCHI_INPUT = 3
  • NOT_ENOUGH_MEMORY = 4
  • ERROR_IN_FLAG_CHAR = 5

See inchi_api.h.
