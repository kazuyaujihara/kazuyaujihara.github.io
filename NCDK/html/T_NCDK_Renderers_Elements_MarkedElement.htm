MarkedElement Class

A marked element adds meta-data (id and tags) to a CDK rendering element (or group of elements). The id should be unique per depiction. The primary use case it to be able to set the 'id' and 'class' attributes in SVG.

Examples
To set the mol, atom, or bond id set a string property to ID_KEY. Similarly, the CLASS_KEY can be used to set the classes.
Copy
IAtomContainer mol;
atom.SetProperty(MarkedElement.ID_KEY, "my_atm_id");
atom.SetProperty(MarkedElement.CLASS_KEY, "h_donor");
atom.SetProperty(MarkedElement.CLASS_KEY, "h_acceptor");
