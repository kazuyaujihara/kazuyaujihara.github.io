AllRingsFinder Class

Compute the set of all rings in a molecule. This set includes every cyclic path of atoms. As the set is exponential it can be very large and is often impractical (e.g. fullerenes).

Remarks
To avoid combinatorial explosion there is a configurable threshold, at which the computation aborts. The threshold values have been precomputed on PubChem-Compound and can be used with the UsingThreshold(AllRingsFinderThreshold) . Alternatively, other ring sets which are a subset of this set offer a tractable alternative.
