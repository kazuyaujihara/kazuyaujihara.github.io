Generate
Generate a SMARTS string from the provided molecule. The generator uses Exprs stored on the {@link QueryAtom} and {@link QueryBond} instances. 
            {@code
            IAtomContainer mol = ...;
            QueryAtom qatom1 = new QueryAtom(mol.Builder);
            QueryAtom qatom2 = new QueryAtom(mol.Builder);
            QueryBond qbond  = new QueryBond(mol.Builder);
            qatom1.setExpression(new Expr(TypeOfExpression.IS_AROMATIC));
            qatom2.setExpression(new Expr(TypeOfExpression.IS_AROMATIC));
            qbond.setExpression(new Expr(TypeOfExpression.IS_ALIPHATIC));
            qbond.setAtoms(new IAtom[]{qatom1, qatom2});
            mol.addAtom(qatom1);
            mol.addAtom(qatom2);
            mol.AddBond(qbond);
            String smartsStr = Smarts.Generate(mol);
            // smartsStr = 'a!:a'
            }
@param mol the query molecule @return the SMARTS @see Expr @see org.openscience.cdk.isomorphism.matchers.IQueryAtom @see org.openscience.cdk.isomorphism.matchers.IQueryBond

Remarks
Given an IAtomContainer a SMARTS pattern is parsed and new IQueryAtoms and IQueryBonds are appended to the connection table. Each query atom/bond contains an Expr that describes the predicate to check when matching. This Expr is also used for generating SMARTS.
Copy
IAtomContainer mol = ...;
if (Smarts.Parse(mol, "[aD3]a-a([aD3])[aD3]")) 
{
    String smarts = Smarts.Generate(mol);
}
