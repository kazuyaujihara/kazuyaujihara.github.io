SmartsPattern Class

Note: This API is now obsolete.

A Pattern for matching a single SMARTS query against multiple target compounds. The class should not be used for matching many queries against a single target as in substructure keyed fingerprints. The SMARTSQueryTool is currently a better option as less target initialistion is performed.

Examples
Simple usage:
Copy
Pattern ptrn = SmartsPattern.Create("O[C@?H](C)CC", ChemObjectBuilder.Instance);

foreach (var ac in acs)
{
    if (ptrn.Matches(ac))
    {
        // 'ac' contains the pattern
    }
}
Obtaining a Mappings instance and determine the number of unique matches.
Copy
Pattern ptrn = SmartsPattern.Create("O[C@?H](C)CC", ChemObjectBuilder.Instance);

foreach (var ac in acs)
{
    nUniqueHits += ptrn.MatchAll(ac).CountUnique();
}
