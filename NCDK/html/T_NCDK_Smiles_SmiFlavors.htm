Canonical Output SMILES in a canonical order. The order is not guaranteed to be equivalent between releases.
InChI
InChILabelling Output SMILES in a canonical order using the InChI labelling algorithm.
AtomAtomMap Output atom-atom mapping for reactions and atom classes for molecules. The map index is set on an atom with property AtomAtomMapping using SetProperty(Object, Object).
AtomicMass Output atomic mass on atoms. For historical reasons the atomic mass is often set on all atoms in a CDK molecule. Therefore to avoid SMILES like "[12CH3][12CH2][16OH]" major isotopes are not generated. If you wish to generate SMILES with the major isotopes please use the flag AtomicMassStrict this will output all mass numbers and only be omitted when the mass is unset (null).
UseAromaticSymbols Writes aromatic atoms as lower case letters. For portability this option is not recommended.
StereoTetrahedral Output tetrahedral stereochemistry on atoms as "@" and "@@".
StereoCisTrans Output cis-trans stereochemistry specified by directional "\" of "/" bonds.
StereoExTetrahedral Output extended tetrahedral stereochemistry on atoms as "@" and "@@". Extended tetrahedral captures rotations around a cumulated carbon: "CC=[C@]=CC".
StereoExCisTrans
AtomicMassStrict Obsolete. Generate SMILES with the major isotopes, only omit mass numbers when it is unset.
Stereo Output supported stereochemistry types.
Cx2dCoordinates Output 2D coordinates.
Cx3dCoordinates Output 3D coordinates.
CxCoordinates Output either 2D/3D coordinates.
CxAtomLabel Output atom labels, atom labels are specified by Label.
CxAtomValue Output atom values, atom values are specified by the property Comment using SetProperty(Object, Object)
CxRadical Output radicals, radicals are specified by GetConnectedSingleElectrons(IAtom)
CxMulticenter Output multi-center bonds, positional variation is specified with Sgroups of the type ExtMulticenter.
CxPolymer Output polymer repeat units is specified with Sgroups.
CxFragmentGroup Output fragment grouping for reactions.
AtomAtomMapRenumber Renumber AtomAtomMaps during canonical generation
CxSmiles Output CXSMILES layers.
CxSmilesWithCoords Output CXSMILES layers and coordinates.
Generic Output non-canonical SMILES without stereochemistry, atomic masses.
Unique Output canonical SMILES without stereochemistry, atomic masses.
Isomeric Output non-canonical SMILES with stereochemistry, atomic masses.
Absolute Output canonical SMILES with stereochemistry, atomic masses.
Default Default SMILES output write Stereochemistry, Atomic Mass, and CXSMILES layers. The ordering is not canonical.
