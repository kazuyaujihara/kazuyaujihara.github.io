InChITautomerGenerator Class

Creates tautomers for a given input molecule, based on the mobile H atoms listed in the InChI. Algorithm described in Thalheim, T. and Vollmer, A. and Ebert, R. and Kuehne, R. and Schuurmann, G., Tautomer Identification and Tautomer Structure Generation Based on the InChI Code, J. Chem. Inf. Model., 2010, 50:1223-1232 .

Provide your input molecules in Kekulé form, and make sure atom type are perceived.

When creating an input molecule by reading an MDL file, make sure to set implicit hydrogens. See the InChITautomerGeneratorTest test case.
