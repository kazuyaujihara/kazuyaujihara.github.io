ChemModelManipulator Class

Class with convenience methods that provide methods from methods from ChemObjects within the ChemModel. For example:
Copy
ChemModelManipulator.RemoveAtom(chemModel, atom);
will find the Atom in the model by traversing the ChemModel's MoleculeSet, Crystal and ReactionSet fields and remove it with the RemoveAtom(IAtom) method.
