public enum MolecularWeightTypes
Members
  Member nameDescription
MolWeight For use with GetMass(IAtomContainer, MolecularWeightTypes). This option uses the mass stored on atoms (ExactMass) or the average mass of the element when unspecified.
MolWeightIgnoreSpecified For use with GetMass(IAtomContainer, MolecularWeightTypes). This option ignores the mass stored on atoms (ExactMass) and uses the average mass of each element. This option is primarily provided for backwards compatibility.
MonoIsotopic For use with GetMass(IAtomContainer, MolecularWeightTypes). This option uses the mass stored on atoms ExactMass or the mass of the major isotope when this is not specified.
MostAbundant For use with GetMass(IAtomContainer, MolecularWeightTypes). This option uses the mass stored on atoms ExactMass and then calculates a distribution for any unspecified atoms and uses the most abundant distribution. For example C6Br6 would have three 79Br and 81Br because their abundance is 51 and 49%.
